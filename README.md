## Features
- **Cyberpunk-Quant UI:** Ultra-dark matte slate backgrounds, neon highlights, and glassmorphism.
- **Zero-Framework SPA:** Built entirely on Vanilla JavaScript, ES Modules, and standard HTML5/CSS3.
- **Custom Charting Engine:** High-performance Canvas API rendering for OHLCV data.
- **Derivatives Analytics:** Real-time Black-Scholes options pricing and Greeks calculations.
- **Pattern Recognition:** Geometric pattern detection and scanning logic.
